# Nested Flexbox

It refers to the use of multiple flex container within each other in HTML & CSS creating complex, responsive & highly flexible layouts. It allows the flex item to be a container for its own child elements.

## Advantages
1) Complex Layout
2) Responsive Design
3) Alignment & Spacing
